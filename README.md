# Williams Alex Portfolio

Personal portfolio website for Williams Alex, a Full-Stack Software Engineer focused on building, maintaining, and scaling web and mobile applications.

Live site: [https://williams-alex-portfolio.netlify.app/](https://williams-alex-portfolio.netlify.app/)

## About Me

Full-Stack Software Engineer with years of experience building, maintaining, and scaling web and mobile applications.

I'm passionate about problem-solving, learning new tools, and creating innovative digital solutions.

## What This Site Includes

- Hero section with intro and resume link
- About section with professional summary
- Skills section grouped by:
  - Programming languages
  - Libraries and frameworks
  - Databases and message queues
  - Cloud and infrastructure
- Experience timeline
- Projects showcase
- Contact form

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Create React App

## Featured Skills

- Frontend: React, Vue, Next.js, Nuxt.js, React Native
- Backend: Laravel, Express.js, Nest.js, FastAPI
- Data: MongoDB, PostgreSQL, Redis, RabbitMQ
- Cloud/Dev: Netlify, Render, Vercel, DigitalOcean, GCP, AWS, Git, GitHub

## Scripts

In the project directory, run:

- `npm start` - Starts the development server at `http://localhost:3000`
- `npm run build` - Builds an optimized production bundle in `build/`
- `npm test` - Runs tests in watch mode
